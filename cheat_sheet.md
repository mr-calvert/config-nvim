# Movement
| keymap | description |
| --- | --- |
| <C-d> | Scroll half page down |
| <C-u> | Scroll half page up |
| <C-n> | Popup next selection |
| <C-p> | Popup previous selection |
| <C-y> | Popup accept current selection |
| <C-e> | Popup quit and close |

# Editing
| keymap | description |
| --- | --- |
| :r | Read the contents of another file in above the cursor |

# Windows
All are off <C-w>... so just start with it and use telescope
| keymap | description |
| --- | --- |
| <C-wv> | Split window vertically |
| <C-wq> | Close the window the cursor is in |
| <C-w[move]> | Select window with movement directions |

# Telescope
| keymap | description |
| --- | --- |
| [leader]<sp> | Open buffers by name/path |
| [leader]sn | nvim configuration files |

# Treesitter
| keymap | description |
| --- | --- |
| :Inspect | Treesitter annotations for cursor location |
| :InspectTree | Split window with treesitter's tag tree centered at cursor |
